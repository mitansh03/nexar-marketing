Create a complete, production-ready, minimalist portfolio website for Nexar Marketing, a digital marketing agency specializing in performance marketing (SEO/SEM, paid social ads) and web development services.

⚠️ PROJECT ARCHITECTURE (VERY IMPORTANT)

Use a monorepo structure with separate frontend and backend folders

Frontend and backend must be completely isolated

Frontend must be pure static output (HTML/CSS/JS or React + Vite)

Backend must NOT interfere with frontend build output

Folder structure:
/client        → Frontend (React + Vite OR HTML/CSS/JS)
  ├── src
  ├── public
  ├── index.html
  ├── vite.config.ts (if React)
  ├── package.json
  └── dist (generated build output)

/server        → Backend (optional)
  ├── index.ts / index.js
  ├── routes
  ├── controllers
  └── package.json


Frontend build output must be /client/dist

Backend build output must be /server/dist

Frontend deployment must work independently (Vercel-ready)
